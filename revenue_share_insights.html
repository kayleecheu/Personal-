Purpose of This Dashboard
This dashboard is a discussion starter, not a final answer. Performance analysis surfaces which players produce value relative to their cap allocation — the deeper question is how that maps to revenue share strategy: which contracts are structurally sound, which tiers are systematically overpaid, and how collegiate investment patterns (NIL, revenue share) are reshaping the pipeline into professional rosters.

Each league uses a different tier system — NBA uses BPM + WS/48 (both required), WNBA uses WS/40 + TS% (efficiency-banded OR logic), and NCAA uses a four-metric composite. Switching leagues in the nav updates every tier reference across the dashboard.
Metric Definitions
BPM — Box Plus/Minus
Estimates a player's per-100-possession contribution above league average. Scale: elite players sit 8–12+, average starters near 0, rotation players −2 to −4. NBA primary metric. Also available for NCAA. Not published for WNBA.
WS/40 — Win Shares per 40 min
Wins contributed per 40 minutes played. Normalizes for playing time. Uses 40-min baseline (vs 48) to match WNBA game length. WNBA primary metric. Strong signal for interior players with high-efficiency, lower-usage profiles.
WS / AAV — Value Metric
Win Shares divided by Annual Average Value (cap hit in $M). Answers the core revenue share question: how much production per dollar of cap? High = surplus value. Rookie deals dominate this metric — filter by Veteran to compare fairly.
